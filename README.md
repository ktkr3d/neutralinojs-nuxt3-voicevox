# neutralinojs-nuxt3-voicevox

## Clone and Build
```
git clone https://github.com/ktkr3d/neutralinojs-nuxt3-voicevox.git
cd neutralinojs-nuxt3-voicevox
neu udate
cd nuxt3-src
npm install
npm run generate
cd ..
neu build --release
```

## Build
```
cd nuxt3-src; npm run generate; cd ..; neu build --release
```

## Run
```
neu run
```
